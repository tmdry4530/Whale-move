# When Whales Move

이 프로젝트는 이더리움 온체인 데이터와 거래소 입출금 데이터를 바탕으로,  
**주요 시장 이벤트 전후에 고래 자금 이동 패턴이 어떻게 달라지는지 분석하고 시각화한 개인 프로젝트**입니다.

핵심은 **데이터 기반 질문 설정, 가설 검증, 시각화**이며,  
뉴스 데이터 수집은 사건의 배경 설명을 돕는 **보조 자료**로 활용했습니다.

---

## 1. 프로젝트 목적

이 프로젝트의 핵심 질문은 다음과 같습니다.

1. **위기 이벤트가 발생하면 거래소 입금량은 증가하는가?**
2. **호재 이벤트에서는 거래소 출금량이 더 강하게 증가하는가?**
3. **한국 이벤트는 글로벌 이벤트보다 변화폭이 작은가?**

이 질문에 답하기 위해 고래 송금 데이터, 거래소 입출금 데이터, 가격 데이터를 결합해  
이벤트 전후의 패턴을 비교 분석했습니다.

---

## 2. 프로젝트 핵심 요약

- 분석 기간: **2017 ~ 2026**
- 이벤트 수: **21개**
- 고래 전송 데이터: **1,307행**
- 거래소 입출금 데이터: **15,393행**
- 가격 데이터: **6,330행**
- 주요 지표:
  - 고래 송금량
  - 거래소 입금량 / 출금량
  - ETH 가격 변화
  - 뉴스 볼륨(일별 기사 수)
  - 이벤트별 전후 7일 반응

## 5. 실행 방법


## 7. 분석 결과 요약

### 가설 1
**위기 이벤트에서는 거래소 입금량이 크게 증가한다.**

- crash 이벤트 평균 거래소 inflow: **+167.4%**

### 가설 2
**호재 이벤트에서는 거래소 출금량이 입금량보다 더 강하게 반응한다.**

- rally 이벤트 평균
  - inflow: **+28.7%**
  - outflow: **+53.6%**

### 가설 3
**글로벌 이벤트가 한국 이벤트보다 온체인 전체 자금 이동에 더 큰 영향을 준다.**

- 글로벌 이벤트 평균 고래 송금량: **+25.3%**
- 한국 이벤트 평균 고래 송금량: **-11.4%**

자세한 내용은 아래 문서를 참고하세요:

## 9. 이 프로젝트에서 중요한 점

이 프로젝트는 단순히 데이터를 모으는 것이 아니라,  
다음 흐름으로 구성했습니다.

> **질문 설정 → 가설 수립 → 데이터 분석 → 시각화 → 결론 도출**

즉, 이번 과제의 핵심인
- 데이터셋 활용
- 질문 도출
- 분석 결과 출력
- 시각화

를 모두 포함하도록 설계했습니다.

---

## 10. 한계점

- 일부 초기 연도 이벤트는 보조 지표 데이터가 부족합니다.
- 뉴스 데이터는 사건 맥락 설명용 보조 자료이며, 분석의 중심은 아닙니다.
- 프론트 번들은 현재 다소 큰 편이라 추가 최적화 여지가 있습니다.
